## How to Contribute

### Reporting Bugs
- Open an issue with a clear description of the bug.
- - Include steps to reproduce, expected behavior, and screenshots if possible.
 
  - ### Suggesting Features
  - - Open an issue with the `enhancement` label.
    - - Describe the feature and its potential use case.
     
      - ### Submitting Pull Requests
      - 1. Fork the repository.
        2. 2. Create a feature branch: `git checkout -b feature/your-feature-name`
           3. 3. Commit your changes with clear messages.
              4. 4. Push to your fork and open a Pull Request.
                
                 5. ## Development Setup
                
                 6. ```bash
                    git clone https://github.com/thierrymaesen/eo-visual-scout.git
                    cd eo-visual-scout
                    pip install -r requirements.txt
                    ```

                    ## Code Style
                    - Follow PEP 8 for Python code.
                    - - Use meaningful variable and function names.
                      - - Add docstrings to all public functions.
